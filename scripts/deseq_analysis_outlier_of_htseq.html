DESeq Analysis of Cushing and Acromegaly Patient Samples with Outlier Removed

The counts tables were generated previously using HTseq.sh shell script and merge.command on Hera. It removes the outlier data point from the analysis, which was patient 29. This script requires a transcript counts table. There is also a sample mapping file called patient_sample_mapping.csv which links the diagnosis the the samples. This file was most recently processed on Sun Jun 1 15:01:31 2014.

Annotation

This step annotates the data tables with the official gene symbols.

The data was annotated from Ensembl data, using the biomaRt package (Durinck et al. 2009; Durinck et al. 2005).
